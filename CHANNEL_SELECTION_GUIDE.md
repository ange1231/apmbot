# 🎯 Выбор каналов для рассылки постов

## 🚀 Новый функционал

Теперь вы можете выбирать конкретные каналы для рассылки постов о ганпаках!

## 📱 Как это работает в админ панели

### 1. Выбор каналов
- Нажмите 📢 у ганпака в списке
- Откроется страница выбора каналов
- Отметьте нужные каналы галочками

### 2. Кастомизация поста
- Добавьте свой текст или используйте автоматический
- Предпросмотр поста в реальном времени
- Выбор всех/снятие всех каналов

### 3. Создание постов
- Публикация только в выбранных каналах
- Детальная статистика результатов
- Graceful fallback для ошибок

## 🎨 Интерфейс выбора каналов

### Визуальные карточки каналов:
```
┌─────────────────────────────┐
│ ✅ Название канала          │
│    @username                │
│    Описание канала...       │
│    [Активен]                │
│ ☑️ Выбрано                  │
└─────────────────────────────┘
```

### Функции управления:
- ✅ **Выбрать все** - отметить все активные каналы
- ⬜ **Снять все** - убрать все отметки
- 👁️ **Предпросмотр** - увидеть как будет выглядеть пост

### Выборочные каналы:
```bash
python manage_posts.py create --gunpack 1 --channels @channel1 @channel2
```

### Все активные каналы:
```bash
python manage_posts.py create --gunpack 1
```

### Кастомный текст:
```bash
python manage_posts.py create --gunpack 1 --text "Специальное предложение!"
```

## 🔧 Технические детали

### Форма данных:
```html
<input type="checkbox" name="channels" value="@channel1">
<input type="checkbox" name="channels" value="@channel2">
<textarea name="post_text">Кастомный текст</textarea>
```

### Обработка на сервере:
```python
selected_channels = request.form.getlist('channels')
post_text = request.form.get('post_text', '').strip()
```

### Асинхронная отправка:
```python
for channel in selected_channels:
    result = await poster.create_post_with_button(channel, gunpack_id, text)
```
