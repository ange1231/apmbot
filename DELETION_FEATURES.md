# 🗑️ Функции удаления и очистки базы данных

## 📋 Что добавлено

### 1. Улучшенное удаление ганпаков
**❌ Было:**
```python
@app.route('/gunpacks/<int:id>/delete', methods=['POST'])
def delete_gunpack(id):
    gunpack = db.query(Gunpack).filter(Gunpack.id == id).first()
    if gunpack:
        db.delete(gunpack)  # Только ганпак
```

**✅ Стало:**
```python
@app.route('/gunpacks/<int:id>/delete', methods=['POST'])
def delete_gunpack(id):
    gunpack = db.query(Gunpack).filter(Gunpack.id == id).first()
    if gunpack:
        # Удаляем все связанные скачивания
        downloads = db.query(Download).filter(Download.gunpack_id == id).all()
        for download in downloads:
            db.delete(download)
        
        # Удаляем сам ганпак
        db.delete(gunpack)
        db.commit()
        
        # Показываем статистику удаления
        downloads_count = len(downloads)
        if downloads_count > 0:
            flash(f'Ганпак успешно удален! Также удалено {downloads_count} записей о скачиваниях.', 'success')
```

### 2. Очистка базы данных от осиротевших записей
```python
@app.route('/admin/cleanup')
def cleanup_database():
    """Очистка базы данных от осиротевших записей"""
    # Находим скачивания без связанных ганпаков
    orphan_downloads = db.query(Download).outerjoin(Gunpack).filter(Gunpack.id.is_(None)).all()
    
    # Удаляем осиротевшие скачивания
    deleted_count = 0
    for download in orphan_downloads:
        db.delete(download)
        deleted_count += 1
    
    db.commit()
```

## 🚀 Как использовать

### Удаление ганпака:
1. Перейдите в `/gunpacks`

3. Подтвердите действие
4. Система удалит все осиротевшие записи

## 🔍 Что такое "осиротевшие записи"?

**Осиротевшие записи** - это записи в таблице `downloads`, которые ссылаются на несуществующие ганпаки.

**Как появляются:**
1. Ганпак удаляется напрямую из БД (минуя приложение)
2. Ошибка при удалении ганпака
3. Восстановление БД из бекапа

**Проблемы:**
- ❌ Битая статистика
- ❌ Ошибки в отчетах
- ❌ Лишние данные в БД

**Решение:**
- ✅ Каскадное удаление при удалении ганпака
- ✅ Периодическая очистка осиротевших записей
- ✅ Проверка целостности данных
