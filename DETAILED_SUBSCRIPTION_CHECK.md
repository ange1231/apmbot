# 🎯 Детальная проверка подписок

## 🚀 Новая функциональность

Теперь бот показывает точно, на какие каналы пользователь не подписан!

## 📱 Как работает

### 1. Умная проверка
Бот проверяет каждый канал отдельно и создает два списка:
- ✅ **Подписанные каналы** - уже выполнены условия
- ❌ **Неподписанные каналы** - нужно подписаться

### 2. Детальный отчет
Пользователь видит:
- Какие каналы уже подписаны ✅
- На какие нужно подписаться ❌
- Кликабельные ссылки для быстрой подписки

## 🔧 Технические изменения

### Функция проверки подписок:
```python
async def check_subscription(user_id: int, channels: list) -> tuple[bool, list]:
    """Проверяет подписку пользователя на каналы, возвращает (статус, список неподписанных)"""
    unsubscribed_channels = []
    
    for channel in channels:
        try:
            # Проверка каждого канала
            member = await bot.get_chat_member(chat_id, user_id)
            
            if member.status not in ['member', 'creator', 'administrator']:
                unsubscribed_channels.append(channel)
        except Exception as e:
            unsubscribed_channels.append(channel)
    
    return len(unsubscribed_channels) == 0, unsubscribed_channels
```

### Обработка результатов:
```python
is_subscribed, unsubscribed = await check_subscription(user_id, channels)

if is_subscribed:
    # Выдаем ганпак
    await message.answer("✅ Отлично! Вы подписаны на все каналы!")
else:
    # Показываем детальную информацию
    text = "❌ Вы не подписаны на следующие каналы!\n\n"
    
    # Неподписанные каналы
    for channel in unsubscribed:
        text += f"❌ [{channel}]({link}) ← не подписан\n"
    
    # Подписанные каналы
    subscribed = [c for c in channels if c not in unsubscribed]
    if subscribed:
        text += "\nПодписаны на:\n"
        for channel in subscribed:
            text += f"✅ [{channel}]({link})\n"
```

## 📊 Статусы подписки

### ✅ Успешные статусы:
- `member` - обычный участник
- `creator` - создатель канала  
- `administrator` - администратор

### ❌ Неуспешные статусы:
- `left` - вышел из канала
- `kicked` - заблокирован
- `restricted` - ограниченные права
