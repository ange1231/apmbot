# 🌙 Полная темная тема для админ панели

## 🎯 Что реализовано

**Все страницы админ панели теперь доступны в темной теме!**

## 📁 Созданные файлы

### Шаблоны:
```
templates/
├── base_dark.html           # Базовый шаблон темной темы
├── dashboard_dark.html      # Главная панель
├── users_dark.html          # Пользователи
├── gunpacks_dark.html       # Ганпаки
├── channels_dark.html        # Каналы
├── gunpack_form_dark.html    # Форма ганпака
├── channel_form_dark.html     # Форма канала
└── statistics_dark.html       # Статистика
```

### Статические файлы:
```
static/
└── logoSblack.png          # Логотип для темной темы
```

## 🎨 Дизайн система

### Цветовая палитра:
- **Фон:** `#1a1a1a` (темно-серый)
- **Карточки:** `#2d3436` (темно-синий)
- **Акценты:** 
  - Синий: `#4a5568` (основной)
  - Зеленый: `#28a745` (успех)
  - Желтый: `#ffc107` (предупреждение)
  - Красный: `#dc3545` (опасность)
- **Текст:** `#ffffff` (белый)
- **Мутный:** `#adb5bd` (серый)

### Градиенты:
```css
/* Сайдбар */
background: linear-gradient(135deg, #2d3436 0%, #000000 100%)

/* Кнопки */
background: linear-gradient(135deg, #4a5568 0%, #2d3436 100%)

/* Карточки */
background: linear-gradient(135deg, #2d3436 0%, #1a1a1a 100%)
```
