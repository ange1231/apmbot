# 🖼️ Руководство по использованию изображений

## 📸 Поддерживаемые сервисы

### ✅ ibb.co (ImgBB)
- **Формат:** `https://ibb.co/xxxxxxx`
- **Автоматически преобразуется:** `https://i.ibb.co/xxxxxxx.jpg`
- **Поддерживаемые типы:** JPG, PNG, GIF, WebP, BMP

### ✅ Прямые ссылки
- **Формат:** `https://example.com/image.jpg`
- **Требования:** Прямая ссылка на файл изображения

### ✅ Другие хостинги
- Imgur
- PostImage
- Любые сервисы с прямыми ссылками

## 🔧 Как использовать

### 1. Загрузка изображения на ibb.co
1. Перейдите на [ibb.co](https://ibb.co)
2. Загрузите изображение
3. Скопируйте URL вида: `https://ibb.co/abc123`
4. Вставьте в поле "URL изображения" в админ панели

### 2. В админ панели
1. Откройте создание/редактирование ганпака
2. Вставьте URL изображения в поле "URL изображения"
3. Сохраните ганпак

### 3. Примеры URL
```
✅ Правильно:
https://ibb.co/abc123
https://i.ibb.co/abc123/image.jpg
https://example.com/image.png

❌ Неправильно:
https://ibb.co/album/abc123 (если это не прямое изображение)
https://example.com/page.html
```

## 🚀 Преимущества ibb.co

- ✅ Бесплатный сервис
- ✅ Быстрая загрузка
- ✅ Автоматическое преобразование URL
- ✅ Поддержка всех форматов изображений
- ✅ Стабильные прямые ссылки

## 🔍 Как работает автоматическое преобразование

Бот автоматически преобразует URL:
```
https://ibb.co/abc123
↓
https://i.ibb.co/abc123.jpg
```

Это позволяет Telegram корректно отображать изображения.

## ⚠️ Важные моменты

1. **Размер изображения:** до 20 МБ
2. **Форматы:** JPG, PNG, GIF, WebP, BMP
3. **URL должен быть доступен** для Telegram
4. **Если изображение не загрузится** - пользователь увидит текстовое описание
