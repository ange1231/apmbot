# Telegram Bot для Ганпаков с Админ Панелью

Полнофункциональный Telegram бот для раздачи ганпаков с системой подписок на каналы и красивой админ панелью.

## 🚀 Функции

### Для пользователей:
- 📦 Просмотр доступных ганпаков
- 🖼️ Красивые карточки ганпаков с изображениями
- ✅ Проверка подписки на каналы
- 🔗 Получение ссылок на скачивание после подписки
- 📱 Удобный интерфейс с кнопками

### Для администраторов:
- 🎛️ Веб-панель управления
- 👥 Просмотр всех пользователей
- 📊 Статистика скачиваний
- 📤 Экспорт пользователей в XML
- ✏️ Редактирование ганпаков в реальном времени
- 🔄 Включение/выключение ганпаков

## 📋 Требования

- Python 3.8+
- Telegram Bot Token
- SQLite (встроен)

## 🛠️ Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd telegram-gunpack-bot
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # для Windows: venv\Scripts\activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл `.env`:
```bash
cp .env.example .env
```

5. Отредактируйте `.env` файл:
```env
BOT_TOKEN=your_telegram_bot_token_here
ADMIN_ID=your_admin_telegram_id_here
DB_PATH=database.db
FLASK_SECRET_KEY=your_secret_key_here
```

6. Обновите каналы в `config.py`:
```python
REQUIRED_CHANNELS = [
    '@your_channel1',
    '@your_channel2',
    '@your_channel3'
]
```

## 🚀 Запуск

### Запуск бота:
```bash
python bot.py
```

### Запуск админ панели:
```bash
python app.py
```

Админ панель будет доступна по адресу: `http://localhost:5000`

## 🎮 Использование

### Для пользователей:
1. Запустите бота командой `/start`
2. Нажмите "🎁 Ганпаки"
3. Выберите интересующий ганпак
4. Подпишитесь на указанные каналы
5. Нажмите "✅ Проверить подписки"
6. Получите ссылку на скачивание

### Для администраторов:
1. Откройте `http://localhost:5000`
2. Добавьте ганпаки через панель управления
3. Следите за статистикой
4. Экспортируйте пользователей при необходимости

## 🔧 Настройка

### Добавление новых каналов:
Отредактируйте `config.py` и шаблон `gunpack_form.html`

### Кастомизация дизайна:
Измените CSS в `templates/base.html`

### Добавление новых полей:
Обновите модели в `database.py` и соответствующие шаблоны

## 📊 Возможности админ панели

- **Главная**: Общая статистика и быстрые действия
- **Пользователи**: Список всех пользователей с экспортом в XML
- **Ганпаки**: Управление ганпаками (создание, редактирование, удаление)
- **Статистика**: Детальная статистика по скачиваниям каждого ганпака

## 🔒 Безопасность

- Админ панель защищена секретным ключом
- Только администратор может управлять ганпаками
- Все данные хранятся локально в SQLite

## 🐛 Устранение проблем

1. **Бот не запускается**: Проверьте токен в `.env`
2. **Админ панель недоступна**: Убедитесь что порт 5000 свободен
3. **Не проверяются подписки**: Убедитесь что бот является админом в каналах
